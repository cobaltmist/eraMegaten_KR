
メアリ口上 Take2

	◆はじめに
メアリ口上書いているうちに色々やりたいことが増えてきたのですが、
それを実装しようとしたらメアリの人の脳が耐え切れなくなったので作り直すことになりました。


	◆ライセンスについて
eraシリーズにおいてのみ運用することを条件に、この口上ファイルのライセンスを放棄致します。
追加・改変・修正・流用・再配布等、特に断らずに行っていただいて構いません。
eraシリーズ以外での利用についてはお断りさせていただきます。


	◆この口上について
「デビルサマナー　ソウルハッカーズ」の登場キャラ、メアリの口上ファイルです。
製作者の趣味により、プレイの内容が若干特殊な性癖に偏った内容になりがちです。
ご利用に際して注意をお願い致します。
また、独自解釈によりメアリのキャラクター性が原作と乖離する場合があります。
メアラーの方は特にご利用に際して注意をお願い致します。


	◆謝辞
愉快なゲームをこれといった見返りもなく作ってくださるＮ鳥さん。

ありがとうございます。

	◆質問を見越したQ & A
Q.主な変更点を教えて
A.一番大きな違いは、恋慕口上が比較的陥落前のメアリと近いものに変わったことでしょうか。
　ちょっとだけ感情を理解している感じにできていると良いんですが。
　他にも細々とした加筆修正新規追加が行われています。

Q.ギミックはどうした
A.従来のギミックの多くはTake2で新規搭載された大型ギミック、感情成長システムと連動する予定です。
　その為、今後の更新にあわせて順次見直した上で搭載されていく予定です。

Q.寝取られはどこいった
A.感情成長システムと連動し、様々なバリエーションでメアリを掻っ攫われるようになります。
　メアリの嗜好によっては寝取らせようとしてもメアリが相手を拒否してしまうこともあります。
　フォローの手段は用意しておくので、まあ大丈夫かなー。

Q.純愛派はどうすんだ
A.元々寝取られは純愛あってこそ＆純愛も好きなので手を抜くつもりはございません。
　ただ、純愛と寝取られは同じ物事の表裏であり、そうなると表面しか使えない純愛よりも、
　裏面まで使える寝取られの方がバリエーション豊かになってしまうことはご了承ください。

○全体共通の仕様
・共通拡張メニュー	「何もしない」を「あなた」が2回連続で実行すると、独自のメニューが表示される。
					内容は「COMPを操作する」。
・COMPを操作する	メアリ口上独自の各種設定を変更可能。
					現在できることは「戦闘口上の表示確率変更」
・経験システム		誰が最初にメアリにそれをさせたかを記録するシステム。
					口上の分岐に影響する。



○恋慕のみの仕様
・会話拡張メニュー	「会話する」を「あなた」が2回連続で実行すると、独自のメニューが表示される。
					内容は「小遣いを与える」
・小遣いを与える	メアリに小遣いを渡すことが出来る。最大￥500まで（条件次第で拡張可能にする予定）。
					小遣いは様々な場面で自動的に使用される。また、一度渡した金銭は回収できない。
