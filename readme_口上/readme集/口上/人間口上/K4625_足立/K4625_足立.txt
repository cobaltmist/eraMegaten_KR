■傾向

＞全体的にキャラクターのイメージを損なう可能性があります
　 「あなた」は名前設定推奨、基本的に常時タメ口です

＞NTR系埋める予定はいまのところありません

・陥落前
　 やさぐれキャベツ（遺伝子組み換えでない）
　 上から目線、死にたくはないので渋々付き合ってやってる

・屈服刻印3
　 若干諦め気味
　 上から目線は変わらないものの状況に流され始める

・崩壊
　 へんじがない　ただのしかばねのようだ
　 ほぼ喋らない、ほとんど反応が返ってこない、されるがまま
　 反発刻印3あると噛みついてきたりする

・恋慕ルート
　 ズッコケデカ（漂白済み）　見かけは人畜無害な感じ
　 未調教陥落かひたすらイチャつこうとするとこうなるはず
　 「あなた」が好きすぎて生きるのが辛い、ヘタレ
　 自分がやらかしたことについては死ぬほど反省している（できれば真っ当な方法で償いたい）

・淫乱ルート
　 マガツ（真っ黒）　目つきがギラギラしてる感じ
　 欲望のままにエロイことし続けるとこうなるはず
　 「あなた」はさておき気持ちいいことが好き、S寄り
　 自分がやったことには後悔も反省もせず開き直っている

・服従ルート
　 犬（漂白失敗）　目に光がない感じ
　 プライド粉砕するぐらい虐め続けるとこうなるはず
　 「あなた」に反逆する発想すら出てこない、ドM（※サド取得時を除く）
　 自分がしたことには罰が与えられるべき（主に「あなた」とP4組から）と思っている

・サド
　 恋慕系陥落…基本は優しくしたいけど「あなた」の悲鳴も好き
　 淫乱系陥落…とりあえず誰でもいいけど嬲りたい
　 服従系陥落…「あなた」が望めばいくらでも虐めます

・マゾ
　 恋慕系陥落…「あなた」になら何されても好き
　 淫乱系陥落…たまにはしてもらうのもいいよね
　 服従系陥落…もはや何されても罰にならないレベル
