・eraMegaten0.309用私家版パッチ纏め_Rev66〜68、あるいはメガネほむら追加パッチ済みの環境でご利用ください。

　暫定的な対応として、同梱のCSVで上書きすることで、旧仕様のメガネほむらに戻り正常に動作するようになります。

・前回淫乱END追加と言っておきながらエンディングが選択できなかった不具合を修正しました。まことに申し訳ございません。

・アナル愛撫実行時の口上を追加しました。
・その他細かいところを修正したり追加したり。
