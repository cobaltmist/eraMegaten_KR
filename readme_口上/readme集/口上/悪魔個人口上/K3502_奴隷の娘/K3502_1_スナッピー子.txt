スナッピー子口上readme

＠この口上はライセンスフリーです。
　改変・再配布もフリーですので、駅前ティッシュだと思ってご自由にお使いください。

＠使い方
　eramegatenERBを開く
　→口上フォルダを開く
　→悪魔個人口上を開く
　→EVENT_K3502_奴隷の娘（スナッピー子）_ALLを投入する
　※名前が似ているので紛らわしいですが、「K3501_あなたの娘」とは無関係のファイルです。

＠内容
　どれが遺伝するかは決まっていませんが、想定した素質は【生意気】【高慢】【狡猾】です。
　息子でも娘でも対応できるように志していますが、若干娘寄りです。

＠注意
　この口上はSM系が充実しています。
　まあ嫌いな方はそもそもコマンドを実行しないから大丈夫だと思いますが、一応。
