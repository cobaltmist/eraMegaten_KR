マザーハーロット専用口上です。

　概要
マザーハーロット専用口上。ver0.02
現行では恋慕関係意外かなりの部分が手付かずです。
よって今回もα版として公開しますん。
誤字修正や日本語修正、加筆、ルート追加何でもビシバシとやっちゃってください。

　
　導入方法
前提条件としてナイトテイルルートのパッチもしくはそれを内包したパッチが必要です。
（マザハ様自体が解禁できないもしくはデータが存在しないので）
カーチャンパッチ（私家版パッチ）などを利用するのが手っ取り早いかもしれません。
パッチを導入したら口上フォルダの悪魔個人用フォルダにERBをシュート！
これにておおよそ動くと思われます。

　コンセプト
前提として真Ⅲなどに登場するマザハ様をメインイメージとしています。
もしかしたら自分の技量的に書けなかったり
性癖的な意味で手を付けられない部分（NTRとか）が出てくるかもなので
ひとまず完成しつつある恋慕だけでも。
「あまあまなあなただけの、あなたのためだけの魔人ちゃんです。」
いっぱい愛でてやりましょう。

　ライセンス
おーるふりー。ご自由にレッツアッセンブル。

バグについて
上記の通り一応動くように（カーチャン環境で確認）してはアリますが、問題が多分に出る可能性はありますので開発スレもしくは本スレに書いて頂ければ多分見ます。
しかし、技量が足りているかは正直自信のない部分なのでボスケテ…といいたい部分も…

予定
やはり未定。衝動的に更新を行うことはあるかもしれませんが、やはり先行き不透明…
よってしばらくはこまごまとコマンドを埋めていくのをメインにしようと思います。


多分書けないので加筆修正希望
