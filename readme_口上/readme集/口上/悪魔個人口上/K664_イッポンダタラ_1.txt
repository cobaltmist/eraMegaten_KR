eraMegaten イッポンダタラ口上

＞始めに
この口上は、アバドン王のイッポンダタラを再現しようとして失敗した口上です。
調教を開始するともれなく女体化したり男の娘になったりするので、嫌いな人は注意してください。

衣装変更はゲンブちゃん口上からコピペさせてもらいました、ありがとうございます。


マッドなメイドを掘ったり掘られたりしたいと思ってこれを作ってみた訳ですが。
調教初回の部分だけ書いて力尽きました……どんな事喋らせればいいのかまったく分からない……
いやまあ、分かりきってた事ですけどね、普通の口上ですら無理だったでしょうし。


代わりに書いてやるよ、という人が居たら嬉しいなーとか思いつつ晒してみます。
