eraMegaten デモンパラサイト 追加パッチ

・内容
当パッチを適用することで、小説版『デモンパラサイト』関連のキャラクター・イベントが追加されます。

・注意
導入前に必ずバックアップを取ってください

当パッチはメガテンシリーズとは無関係ですので、そうしたものに嫌悪感を抱く方は
パッチを導入しないか、ゲーム開始時のコンフィグでEXTRAイベントの発生をなしにしてください。

当パッチはいわゆるα版であり、イベントやキャラクターの追加、バランス調整等が予定されています。
また、将来的に内容に大きな変更も起こりえますことをご了承ください。

・開発環境
本体バージョン0.309 ＋ 私家版パッチ纏め_Rev70 ＋ 私家版パッチ纏め_Rev72 ＋ 私家版パッチ纏め_Rev73
※上記以外の環境で導入した場合、競合・エラーを引き起こす可能性がありますのでご注意ください。
