白き鋼鉄のX追加mod ver1.1.0
　作った人：木綿豆腐

ライセンスフリーです
「俺の方がもっとよく出来るぜ！！！」って人は力を貸してくれー

＊前提環境
eraMegaten0.309＋0.309用修正6+私家版パッチ纏め_Rev132+Rev137

＊導入の仕方
ファイルを放り込むだけ

＊イベント開始条件
EXTRAイベントをONにしていて
あなたのLV50以上で
貢献度777500以上だと単発イベント「白き鋼鉄」が出ます

以後は個別イベントです
解放条件は記述してあるので割愛

＊概要
・個別イベント5『祈歌－プレイヤー－』を追加
　→『碧色の雷霆』クリア後ならいつでも挑戦可能な腕試しステージ兼アキュラ・ＧＶ強化クエストです
　　クリアするとアキュラ・ＧＶに専用装備のアクセサリが解放されます
　　（なんか装備画面で出ないこともあるので、何回か切り替えてみてください）

・一部の顔グラを微調整。誰のどこが変わったか、果たして気づくでしょうか

・暇な間に作った未使用の顔グラも入ってます。そこら辺のランダムキャラとかに指定してみてもいいかもですね
