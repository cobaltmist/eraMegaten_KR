白き鋼鉄のX追加mod ver1.0.3
　作った人：木綿豆腐

ライセンスフリーです
「俺の方がもっとよく出来るぜ！！！」って人は力を貸してくれー

＊前提環境
eraMegaten0.309＋0.309用修正6+私家版パッチ纏め_Rev132+Rev136+eraMegaten_5487(ナコルル専用装備化+専用装備修正)
　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　↑必須です

＊導入の仕方
ファイルを放り込むだけ
(個別イベントにも戦闘イベントを追加するパッチを未導入の方はCSVにあるFlag(差分).csvの内容をFlag.csvにコピペしてください)
※BATTLE.ERB, SYSTEM_BATTLE_DAMAGEに手が及んでいるので注意

＊イベント開始条件
EXTRAイベントをONにしていて
あなたのLV50以上で
貢献度777500以上だと単発イベント「白き鋼鉄」が出ます

以後は個別イベントです
解放条件は記述してあるので割愛

＊概要
・個別イベント4『碧色の雷霆』を追加
　→白き鋼鉄のX END到達後ならいつでも挑戦可能な腕試しステージ兼ブレイド強化クエストです
　　クリアするとブレイドに専用装備としてアクセサリが解放されます

・ダークネストリガー発動中の選択EXウェポンも色分け

・X-ROSS DISASTERの威力を修正

・避雷針で表示する属性を電撃に変更
　→攻撃を実行するのは雷撃鱗のため

・全体的にボスの難易度を調整

・アンカーネクサス、ダークネストリガー発動中の解説が間違っていたので修正

・ラスボスのcsvにEXTRA出典フラグを追加

・クロスランサー実行時エラー落ちする不具合を修正

・以前アップした「個別イベントにも戦闘イベントを追加するパッチ」の内容を含みつつ、SONG OF DIVAの被ダメ処理を修正　←NEW

・シアンの顔グラを地味に修正　←NEW
