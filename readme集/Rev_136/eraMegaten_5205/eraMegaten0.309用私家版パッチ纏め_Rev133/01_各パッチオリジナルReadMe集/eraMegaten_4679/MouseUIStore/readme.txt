マウスUIストア

ジャンクショップと魔貨交換のアイテム交換用の新しいUIを追加します。オプションで切り替えできます。
ペルソナ使いなどを判別せず、属性表示もおおざっぱで低機能ですが手軽さを重視しています。

対象環境
カーチャンRev127
フォント「ＭＳゴシック」かつフォントサイズ・一行の高さ16以下
デフォルト以上の解像度を推奨

使い方
ファイルを導入し、オプションで「（グローバル）RPGアイテム購入画面」を切り替えてください。
また、オプション設定の都合上、先日アップしたOPTIONパッチファイルを含みます。ご了承ください。

仕様
・ジャンクショップは行動メニューを含みます。魔貨交換はアイテム交換メニューのみです。
・ジャンクショップでの売却メニューがJPと円で二つに分かれます。
円での売却は「貴重品を売る」、JPでの売却は「不用品を下取りしてもらう」になります。
・アイテムリストの操作
項目名をクリックでソートします。防御相性のみソートではなく表示色を変更します。
左にある矢印でアイテムをカートへ入れたり戻したりします。増減量は「ＵＮＩＴ」で設定します。
アイテム名などのクリックでカーソルを移動します。さらにクリックすると現在選択中のユニット分購入できます。
アイテムクリックでの購入時のみ、比較しているキャラクタに装備させるかのダイアログが出ます。
・上部メニュー
「ＭＯＮＥＹ」使う通貨を選択します。
「ＵＮＩＴ」購入・売却の量を選択します。
「ＴＹＰＥ」アイテムのカテゴリを選択します。
「ＵＳＥＲ」装備との比較、各種パラメータの計算に使用するキャラクタを選択します。
・下部メニュー
「ＣＡＲＴ」[ＣＬＥＡＲ]ボタンでカートをクリアします。
「ＰＲＩＣＥ」[ＢＵＹ]ボタンなどで購入などの確認に進む、もしくは[ＣＡＮＣＥＬ]ボタンで退出します。

注意点
・状態異常耐性は画面外にはみだします。仕様です。
・スペースの都合上、リスト・比較共に主要な能力変化のみ表示しています。
