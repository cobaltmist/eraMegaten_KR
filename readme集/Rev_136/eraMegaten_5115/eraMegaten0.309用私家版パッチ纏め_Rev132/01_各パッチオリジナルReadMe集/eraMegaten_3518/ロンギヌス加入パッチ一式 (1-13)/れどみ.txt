おまけに13人陥落させてもED文中じゃ誰一人出てきたりしません。武断や娘ハーレムみたいな面白いもんでもねえなこれ？！
それでも「おっしゃちょっと行ってくるわ」みたいな奇特な方がいらしたら上層部への謁見断ってみて、
行けたか駄目だったかご報告お願いします
最低4週でしかも１〜４の子は難易度高い素質とか振ってますがね！

達人の運用がどんなもんかいまいち把握してないので、塩梅わからないのですが、

まさかバク吐いてあわてて修正あげたらパスワード入力間違ってＤＬできなくなるとか…

ふと思いついたけどロンギヌス総勢十三名、陥落させたらペルソナ２特殊エンドとか需要あるんでしょうか

所感とか（15/04/19）

気が向いたのでロンギヌスの続投加えました
前に「１０番の子が命乞いして来た」とだれか言ってたのでとりあえずそれっぽく
作った人間としてこの娘はどういうキャラ付けかっていう方向性はぼんやりとあるけれど
姿形すらプレイするあなたによって異なるので、まぁうまい具合に想像してやってください

ボススキルのＭＧ３４を装備品にするのはちょっとまずいかなーって思ってたけど
他の方に修正いただいて、うまい具合に扱われているとは思うのですが
一応制作者としての意図とちとズレがあったので

どうでもいいけどあのパワードスーツ「ヒンメル・フォイアー」て言うんですね…
