それぞれの説明
●ダンジョン：原宿
　依頼：デビルチルドレン捕獲依頼の内容をダンジョンスタート時に入れデビルチルドレンを選ぶようにし、初期ダンジョンにしました。
　先に闘技場のデビルチルドレンをクリアした場合もう一方のデビルチルドレンが選択されるようにしてあります。
　刹那ルートではマンション（戦闘）、公園、神社（刹那戦）の順で、未来ルートではマンション、学校（戦闘）、神社（未来戦）の順でイベントが進行します。
　地下街にターミナルがあるのでそこを中心にプチフロスト→ジャックフロスト、ピクシー→ハイピクシーにすると楽にです

●依頼：反乱軍からの使者
　メシアのツノ（ヒトミ）を集めることが魔界を旅する理由だとフェンリルが説明します。

　ヘルに捕まりダイアナたちと一緒に脱出し、城にいるヘルを倒しに行くことになります。
