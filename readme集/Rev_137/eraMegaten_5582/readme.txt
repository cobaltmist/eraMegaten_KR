＊内容
eramegatenに以下の修正を加えるパッチです。
・八尺様のEXTRA출전フラグを追加。

・フリーズアームズの文言が間違っていたので修正。	火炎→氷結
・ショックアームズの文言が間違っていたので修正。	火炎→電撃
・ダークエーテルを戦闘使用時に使用文を出すように。

CSV、ERBをそのままいれてください
