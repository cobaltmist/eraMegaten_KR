●各Ver.の詳細
・Ver.4.3
　姫宮菊枝専用スキル「菊理媛の組紐」は最初に独立のスキルとして実装しましたが(SKILL1339)
　オリジナルスキルの氾濫に繋がることを懸念して既存のスキルを変化させる形にしました。
　故にヤマタイカパッチの作者としては、専用のオリジナルスキルを別途追加するのは遠慮していただけると幸いです。。
　
　一方で、Ver.3で実装したスキルが「使いにくい・分かりにくい・調教経験消費はめんどくさい」との意見がありましたので
　組紐の装備要求レベルを1に（最初から使用可能）、スキル変化をダメージ追加（調教能力で威力上昇、消費無し）の形に統一しました。
　>>強すぎるとの意見を元に再調整（調教能力による威力上昇を削除）
　　;従順・欲望・技巧を全て10以上にして得られるのが魔力型シフター専用の一般スキルよりも少し強い程度にした場合、明らかに手間と利益が釣り合っていないが
　　;無暗にＭＰ消費を増やすのもLv1から使用可能にした意味がなくなるため、調教による威力上昇を削除する形で調整
