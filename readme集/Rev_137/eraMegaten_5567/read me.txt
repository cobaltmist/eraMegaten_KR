個別イベントにも戦闘イベントを追加するパッチ

作った人：木綿豆腐

前提：
eraMegaten0.309＋0.309用修正6+私家版パッチ纏め_Rev132＋Rev136+eraMegaten5487+eraMegaten+5550

導入方法：
※BATTLE.ERBにも手が及んでいるので注意
(1)ファイルを本体にコピペ
(2)CSVにあるFlag(差分).csvの内容をFlag.csvにコピペ
以上

やったこと：
個別イベントから実行する戦闘において、戦闘中イベントを呼び出せるようにした。
具体的には以下の５つを実装した。
○ゲームオーバー時イベント
EVENT_INDIVIDUALEVENT_LOSE_K{FLAG:개별이벤트캐릭}
○戦闘開始時イベント
EVENT_BATTLE_START_INDIVIDUALEVENT_K{FLAG:개별이벤트캐릭}
○ターン開始時イベント
EVENT_BATTLE_TURNTOP_INDIVIDUALEVENT_K{FLAG:개별이벤트캐릭}
○ターン終了時イベント
EVENT_BATTLE_TURNEND_INDIVIDUALEVENT_K{FLAG:개별이벤트캐릭}
○行動終了後イベント
EVENT_BATTLE_ACTEND_INDIVIDUALEVENT_K{FLAG:개별이벤트캐릭}

使い方：
FLAG:개별이벤트캐릭には個別イベントを実行するキャラの番号、FLAG:진행중개별이벤트に何番目のイベントかを記録している。
千枝の例）
@EVENT_BATTLE_TURNEND_INDIVIDUALEVENT_K4560
SELECTCASE FLAG:진행중개별이벤트
CASE 0
	（具体的な処理）
CASEELSE
ENDSELECT
このように書くことで、千枝の個別イベントの、0番で戦闘を行う場合、ターン終了時のイベントを実行できる。

同梱のアキュライベントでも使用しているものがあるので参考にどうぞ。
